# Mobile App

Android client for travelers who want to view public reserve alerts and submit reports with optional media attachments.

## Features

- Load public reserves from the backend
- Display published traveler-facing reserve events
- Submit traveler reports with photos or videos
- Uses `10.0.2.2` so an Android emulator can reach the local backend on the host machine

## Run

1. Open `mobile-app` in Android Studio.
2. Start an emulator.
3. Run the `app` configuration.

The backend must already be running on `http://localhost:8080`.
